# 🚀 학사경고자 설문 전문 초고속 입력기 (Survey Data Entry Tool)

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows%20x64-lightgrey.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

마우스 조작 없이 **오직 키보드(Numpad)만으로** 대량의 PDF 스캔 설문지를 초고속으로 전산화할 수 있는 Windows 전용 데이터 입력 솔루션입니다. 

기존 수작업 대비 **압도적인 속도 향상**과 **휴먼 에러(오타, 학번 밀림) 방지**에 초점을 맞추어 개발되었습니다.

## 💻 사용 방법 (How to Use)

### 1. 사전 준비
* 배포된 압축 파일을 다운로드하고 압축을 해제합니다. (※ `x64` 폴더와 `.exe` 파일이 항상 같은 곳에 있어야 합니다.)
* `Form1.exe`를 실행합니다.

### 2. 작업 흐름
1. **명단 로드:** 프로그램 시작 시 나타나는 창에서 학생 명단(`CSV`)을 선택합니다.
2. **PDF 로드:** 스캔된 설문지(`PDF`) 파일을 선택합니다.
3. **데이터 입력:** * 학번 입력 ➔ `Enter`
   * 설문 선택 번호 연속 입력 (예: `1249`) ➔ `Enter`
   * *데이터가 저장되며 사진이 자동으로 다음 장으로 넘어갑니다.*
4. **엑셀 추출:** 모든 입력이 끝나면 좌측 하단의 `최종 엑셀 추출` 버튼을 누르고, 빈칸이 있는 원본 엑셀 템플릿 파일을 선택합니다.

## ⌨️ 단축키 안내 (Hotkeys)

| 키(Key) | 기능(Action) |
| :---: | :--- |
| `Enter` | 다음 입력칸으로 이동 / 데이터 최종 저장 및 다음 사진 넘기기 |
| `ESC` | 현재 화면에 떠 있는 PDF 페이지 건너뛰기 (버리기) |
| `마우스 좌클릭` | 클릭한 PDF 페이지 건너뛰기 (버리기) |

## ✨ 핵심 기능 (Features)

* ⚡ **No-Mouse 초고속 Numpad 입력:** 학번과 사유 번호만 연속으로 치고 `Enter`를 누르면 즉시 다음 장으로 넘어가는 논스톱 프로세스.
* 🛡️ **실시간 학번-성명 교차 검증:** 학생 명단(CSV)을 미리 불러와, 입력한 학번과 실제 PDF 상의 이름이 일치하는지 실시간으로 대조합니다. (명단에 없는 학번 입력 시 오타 경고)
* 📄 **내장형 PDF 뷰어 (Pdfium):** 외부 프로그램 없이 PDF 파일을 직접 로드하여 1장씩 크게 보여줍니다.
* ⏭️ **광속 스킵 기능:** 불필요한 페이지(표지, 안내문 등)는 `ESC` 키나 마우스 클릭 한 번으로 즉시 건너뜁니다.
* 💡 **특수 상태 단축키 지원:** * `9` 입력 ➔ "연락두절" 자동 기입
  * `0` 입력 ➔ "확인요망" 자동 기입
* 📊 **엑셀 자동 매칭 추출:** 입력이 끝난 후 원본 엑셀 템플릿을 선택하면, 해당 학생의 행을 자동으로 찾아 `O` 표시 및 기타 의견을 기입하여 새 파일로 추출합니다.
